== Description ==

Fortpolio is a generic portfolio plugin to add multiple media files (image, video, audio, file) to a custom-post-type. You can adjust the custom-post-type in terms of names, taxonomies, metadata etc.

= Overwriting templates =

To overwrite templates simply create a folder named 'fortpolio' inside your theme folder. The templates you can overwrite are inside the 'template' directory of the plugin folder.

== Installation ==

1. After download place the files in the Wordpress plugin directory (wp-content/plugins/Fortpolio/)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In the admin settings menu you can adjust Fortpolio according to you needs.
